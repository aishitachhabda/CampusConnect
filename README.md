
#  CampusConnect

CampusConnect is a modern web platform designed to bring college students together in one unified digital space. It enables students to connect, collaborate, explore opportunities, and stay updated with everything happening on campus.

---

##  Overview

CampusConnect acts as a **centralized student hub** where users can:

* Discover and register for campus events
* Connect with peers and explore profiles
* Join clubs and communities
* Share posts, updates, and resources

It blends **social networking + productivity tools** tailored specifically for college life.

---

## ✨ Features

###  Authentication

* Secure user signup and login
* Role-based access (Student / Admin)

---

### 🧑‍🎓 User Profiles

* Personalized profiles with:

  * Name, branch, year
  * Skills & interests
  * Profile photo and bio
* Explore other student profiles

---

### 📅Events

* View all upcoming campus events
* Event details: title, description, date, time, venue
* Register for events
* Admins can create and manage events

---

### 💬 Community Feed

* Create posts (text + images)
* Like and comment on posts
* Dynamic and engaging feed

---

### 👥 Clubs & Communities

* Discover and join clubs
* Each club includes:

  * Description
  * Members
  * Posts and discussions

---

### 📚 Study Resources

* Upload and share notes (PDFs, files)
* Categorized by subjects
* Easy search functionality

---

### 🔔 Notifications

* Event reminders
* Club updates
* Activity alerts

---

## 🎨 Design Philosophy
